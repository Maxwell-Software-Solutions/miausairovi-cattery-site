## 🎯 Key Concepts

### What is Keystatic?

A Git-based CMS that:

- Stores content as Markdown files
- Provides visual admin interface
- Maintains type safety
- No vendor lock-in
- Free and open-source

### Why Keystatic?

- ✅ Non-developers can edit content
- ✅ Visual editor (no coding needed)
- ✅ Git-based (version controlled)
- ✅ Type-safe (TypeScript)
- ✅ Static generation (fast)
- ✅ No database needed

### When to Use?

✅ **Yes** - Frequent content updates, non-tech editors
⚠️ **Maybe** - Moderate updates, mixed team
❌ **No** - Rare updates, developer-only team

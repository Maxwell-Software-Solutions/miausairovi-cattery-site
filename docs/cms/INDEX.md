### What is Keystatic?

A Git-based CMS that:

- Stores content as Markdown files
- Provides visual admin interface
- Maintains type safety
- No vendor lock-in
- Free and open-source

### Why Keystatic?

- âœ… Non-developers can edit content
- âœ… Visual editor (no coding needed)
- âœ… Git-based (version controlled)
- âœ… Type-safe (TypeScript)
